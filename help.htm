Press Circle to remove the nav-bar then use the analog stick to control the cursor. You can open an application or game from the desktop or the start menu. To open an application from the desktpo, move the cursor over the icon and press the X button on your PSP. To open an application from the start menu, press X on Start and press X on the application you wish to open.

Maximize - The maximize button () can be pressed so the current game takes up the entire screen, instead of being in a window.

Minimize The minimize button () allows you to hide the window. It can be opened again by clicking on the tab () in the taskbar.

Close - Press in the top-right corner to close an application or game.
